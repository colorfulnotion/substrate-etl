### Blocks
```
SELECT date(block_time) as logDT, MIN(number) startBN, MAX(number) endBN, COUNT(*) numBlocks FROM `substrate-etl.polkadot.blocks0`  where LAST_DAY(date(block_time)) = "2023-02-28" group by logDT order by logDT
```


### Signed Extrinsics

```
