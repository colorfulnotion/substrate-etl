[Schema](https://github.com/colorfulnotion/substrate-etl/blob/main/schema/blocks.json)

```bash
SELECT date(block_time) as logDT, MIN(number) startBN, MAX(number) endBN, COUNT(*) numBlocks 
 FROM `substrate-etl.kusama.blocks0`
```
